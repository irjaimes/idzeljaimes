Recess

Fullstack Application

Recess is fullstack application that keeps track of the user's weekly availability so it can randomly schedule the user's extra-curricular activities. The user must create an account to login to the application. Once logged in, sleep and work hours must be entered to calculate free time. Then, the user is able to select the activity category and add activity to their weekly schedule. All that is needed is, how long the user wants to do the activity, and a description of the type activity. The application will then randomly schedule the added activity to the user's schedule.

Technologies
  • HTML
  • CSS + Bootstrap + Sass
  • Javascript + jQuery
  • Node.js + RESTful APIs
  • Express + Handlebars.js
  • Mysql + Sequelize + DBJaws
  • Deployed through Heroku

Who You Gonna Call

Frontend Application

The purpose of this application is to serve as tool that provides important information, as well as contact information for local representatives. A user could search their area data by simply entering their zip code. If the user is not happy with the numbers, or sees that there's not much involvement of the local government to resolve the increase in cases, they can contact their local representative to request more information.

This application uses server-side APIs to display local Covid-19 data and local representative information. It uses local storage to save previous search queries, and it's mobile responsive.

Technologies
  • HTML
  • CSS + Bulma
  • Javascript + jQuery
  • Deployed through Github
